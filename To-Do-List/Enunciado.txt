1. Crea una página web para organizar una lista (puede ser de la
compra, videojuegos, etc) donde los datos son persistentes y
permanentes usando localStorage:
a. Un botón donde podemos almacenar (setItem) los
nombres, éstos deben aparecer en una lista
b. Un botón “mostrar contenido” para mostrar todos los
valores almacenados en localStorage (JSON.stringify).
c. Eliminar elementos (delete) para poder borrar
elementos de forma individual por su nombre
d. Limpiar el contenido (clear) para eliminar todos los
elementos de la lista

Nota: Separa el estilo y el script en páginas diferentes al HTML ya
que es necesario para el siguiente ejercicio.

2. Crea un archivo manifest para que la página pueda funcionar
aunque no haya conexión. Usa XAMPP u otra aplicación para
tener una conexión y probar su funcionamiento.
